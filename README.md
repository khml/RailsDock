# Get Started

```
# 1. rails new
$ ./rails_new.sh

# 2. Setting config/database.yml
# Editing config/database.yml like following

# default: &default
#   adapter: mysql2
#   encoding: utf8
#   host: db
#   username: root
#   password: root
#   pool: 5


# 3. Build Docker image
$ docker-compose build

# 4. start containers
$ docker-compose up -d

```
